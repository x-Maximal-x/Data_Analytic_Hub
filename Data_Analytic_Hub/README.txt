Data Analytics Hub

IDE: We developed this project using Eclipse IDE

Java Version: The project was built using JavaSE 20.

JavaFX: This project uses JavaFX-17.0.8 for the graphical user interface.

MySQL Database: The project connects to a MySQL Version 8.0.34 database for user authentication . 


Installation and Running the Code


Clone this repository to your local machine or download the source code as a ZIP file and extract it.



Update the database connection details in the DBUtils.java file to match your MySQL configuration.
Make connection using the "jdbc:mysql://localhost:3307/users", "root", "admin" 
configure the database to match the requirement 

Setup login Table with column user_id, username, paswword, firstname and lastname .

Open the project folder using your preferred Java IDE (e.g., Eclipse).
Configure JavaFX dependencies

In Eclipse, you can do this by adding the JavaFX libraries to your project's build path.

Locate the Main.java file in the project directory and run it as a Java application. This will start the Data Analytics Hub application.
